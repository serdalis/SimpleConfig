Detailed Description

Global Configuration File Handler.

Designed to allow flexable allocation of .ini structured configuration files.
This class was designed to provide extendable configuration file reader which could be used at any location within a program.
Adding this class will allow access to the global dictionary of configuration files and configuration Parsers.
With the help of configurable parsers this class allows all types of configuration entries to be read and parsed in any way.
